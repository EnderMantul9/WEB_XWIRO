# **App Name**: Statistika Terpercaya

## Core Features:

- Data Fetching from Firestore: Fetch data from Firestore collections (indicators, statistics_tables, news, publications) using React hooks.
- Chart Generation: Generate line, bar, and pie charts using Chart.js, populated with data from Firestore.
- Homepage Indicators: Display key statistical indicators in a carousel format on the homepage, pulling data dynamically from Firestore.
- Latest Information Display: Showcase the latest statistical releases (news, tables, publications, infographs) in a tabbed section, linking to detailed views.
- News and Press Releases Grid: Display news and press releases in an image grid format with hover effects, linking to full articles.
- Search functionality: Implement a search bar that filters the available statistics.

## Style Guidelines:

- Primary color: Dark blue (#1A3E66) for a professional and institutional feel.
- Background color: Light gray (#F0F0F0), providing a clean backdrop.
- Accent color: Light blue (#ADD8E6) to highlight key elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, formal, and readable look. Note: currently only Google Fonts are supported.
- Use simple, professional icons to represent data categories and types.
- Implement a sticky navbar to improve navigation and user experience.
- Use subtle fade and slide animations for transitions and loading effects.
